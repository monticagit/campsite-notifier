    One-time GitHub setup
    Step 1 — Push the code to GitHub
    Create a new repo at github.com/new, then:
    cd ~/Claude/campsite-notifier
    git init && git add . && git commit -m "init"
    git remote add origin https://github.com/YOUR_USERNAME/campsite-notifier.git
    git push -u origin main

    Step 2 — Add Gmail secrets
    In your GitHub repo → Settings → Secrets → Actions → New repository secret
    Add two secrets:
      • GMAIL_ADDRESS = your Gmail address
      • GMAIL_APP_PASSWORD = your Gmail App Password

    Step 3 — Enable Actions
    Go to the Actions tab in your repo → click "Enable GitHub Actions"

    That's it! 🎉 GitHub runs the checker every 15 min automatically.
